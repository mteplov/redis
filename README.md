### Задание 1. Кеширование 

Приведите примеры проблем, которые может решить кеширование. 

*Приведите ответ в свободной форме.*

Кеширование помогает решить ряд типовых проблем:
1. Большая нагрузка на базу данных
Если одно и то же значение читают сотни/тысячи раз, кеш снимает нагрузку с БД, так как данные читаются из памяти (в разы быстрее).
2. Медленные вычисления
Если результат операции тяжёлый (например, сложные отчёты, расчёты, большие JOIN), кеш позволяет не выполнять вычисления повторно.
3. Высокая задержка при обращении к внешним API
Ответы от внешних сервисов (курсы валют, погода, цены) можно кешировать, уменьшая задержки и количество запросов.
4. Медленная загрузка страниц веб-сайта
Кеширование HTML или частей страницы ускоряет отдачу контента.
5. Срыв работоспособности при большом трафике
Кеширование предотвращает падение сервисов из-за резкого наплыва запросов.
6. Уменьшение стоимости ресурсов
Меньше запросов к БД → меньше нагрузки → можно использовать более слабое железо.
